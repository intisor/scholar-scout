how to set up Notion

1
Go to notion.so/my-integrations and create a new Internal integration. Name it "Opp Tracker". Copy the token above.
2
Create a Notion database with these properties: Name (title), URL (url), Status (select), Final Deadline (date), Next Action Date (date), Organization (text), Share with Juniors (checkbox), Date Clipped (date).
3
Open that database, click "..." → Connections → Add your integration. Then copy the database URL — your DB ID is the 32-char string in it.
4
Paste the token and DB ID above, hit save, and you're done. Go clip something.
